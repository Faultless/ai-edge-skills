Food Macro Tracker

Analyze food from a photo or ingredient list to estimate calories, protein, fat, and carbs using the Spoonacular API. Send a picture of your meal or type your ingredients, and the on-device Gemma model will identify the food and call the Spoonacular API to return a full nutrition breakdown.

JS API Webview Secret Camera

How to use

  1. Open the AI Edge Gallery app and enter the Agent Skills use case.
  2. Tap the Skills chip, then the (+) button, and select Load skill from URL.
  3. Enter the URL pointing to the skill folder, e.g.
    https://<username>.github.io/ai-edge-skills/food-macro-tracker
  4. When prompted, enter your Spoonacular API key.
  5. Take a photo of food or type ingredients like "2 cups rice, 1 lb chicken breast, 1 tbsp olive oil".
